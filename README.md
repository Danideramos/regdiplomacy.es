# Rania El Ghauche · Diplomacy & Strategy

Sitio web estático de una sola página listo para GitHub Pages o cualquier alojamiento estándar.

## Archivos

- `index.html`: estructura de la página y texto definitivo.
- `styles.css`: diseño adaptativo, maquetación, colores y tipografía.
- `script.js`: menú para móviles, servicios en acordeón y animaciones de revelación.
- `assets/` — logotipo e imágenes optimizados.
- `.nojekyll` — evita problemas de procesamiento en GitHub Pages.

## Publicación en GitHub Pages

1. Crea o abre el repositorio para la página web.
2. Sube todos los archivos de esta carpeta a la raíz del repositorio, no dentro de una carpeta adicional.
3. En GitHub Pages, publica desde la rama principal y la carpeta raíz.
4. En GitHub Pages, configura el dominio personalizado como `regdiplomacy.es`.
5. El archivo `CNAME` ya está incluido con `regdiplomacy.es`.
6. Configura el DNS en el proveedor de dominio o en Cloudflare utilizando los registros solicitados por GitHub Pages.
